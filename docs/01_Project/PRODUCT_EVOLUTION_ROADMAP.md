# 产品长期演进路线图（1 → 10，合规版）

## 定位

本文档说明：**这个产品如何在不改变初心、不踩红线的前提下，逐步变"更深、更稳、更有价值"**。面向创始人、核心合伙人、长期规划讨论。

**合规边界**：本系统为行为结构测评与陪跑辅助，不提供投资建议；详见 [`PRODUCT_BOUNDARY_AND_COMPLIANCE.md`](./PRODUCT_BOUNDARY_AND_COMPLIANCE.md)。

---

## 当前所处阶段（1.0 的真实状态）

### 已具备的核心能力

**交易行为结构测评（fast / pro）**：
- fast 测评：9 题，输出主画像（6 种 archetype）与稳定度（high/medium/low）
- pro 测评：18 题，输出六维结构（0-100 归一）与阶段判断（pre/mid/post）
- 测评结果存储在 `attempts` 表，不可修改（`submittedAt` 一旦写入）
- 系统支撑：`lib/scoring.ts`、`data/seed/quiz_fast_v1.json`、`data/seed/quiz_pro_v1.json`

**助教陪跑后台（轻 CRM + SOP）**：
- 轻 CRM：客户管理、邀请管理、标签管理（`app/api/coach/customers/route.ts`、`app/api/coach/invites/route.ts`）
- SOP 实时面板：基于 `stage` + `tags[]` 匹配沟通策略（`lib/sop-matcher.ts`）
- 权限边界：只能查看自己名下客户（`lib/authz.ts` 的 `requireCoachOwnsCustomer`）
- 系统支撑：`app/coach/*`、`app/api/coach/customers/[id]/route.ts`

**数据资产雏形（Attempt / tags / stage / SOP 命中）**：
- Attempt 时间线：多次测评形成行为演进记录（`attemptTimeline`）
- 标签系统：系统标签（`image:*`、`stability:*`、`phase:*`）+ 助教标签（`coach:*`）
- 阶段判断：pro 测评输出阶段（pre/mid/post）
- SOP 命中：基于 `stage` + `tags[]` 匹配 SOP，生成 `realtimePanel`
- 系统支撑：`prisma/schema.prisma`（Attempt、CoachTag、SopDefinition、SopRule）

### 主动放弃的方向

**投资建议**：
- 不提供投资建议
- 不提供资产配置方案
- 不提供买卖决策
- 系统不存储任何收益数据（`prisma/schema.prisma` 中 `Attempt` 表不包含收益字段）

**信号工具**：
- 不输出交易信号
- 不提示买卖时机
- 不推荐标的（个股/基金/期货等）
- 系统不提供任何交易相关功能

**收益导向模型**：
- 不预测交易结果
- 不预测收益或回报
- 不承诺收益或回报
- 系统不提供任何收益预测功能

- fast 测评输出主画像与稳定度

- 系统能够识别行为结构，但解释能力有限

**演进方向**：
- 更细致的行为解释层
- 更清晰的结构反馈方式
- 更好的阶段性复盘支持

## 整条路线的"自我约束原则"

### 永远不跨越的边界

**不提供投资建议**：
- 不提供投资建议
- 不提供资产配置方案
- 不提供买卖决策

**不提供交易信号**：
- 不输出交易信号
- 不提示买卖时机
- 不推荐标的（个股/基金/期货等）

**不预测结果**：
- 不预测交易结果
- 不预测收益或回报
- 不预测胜率或成功率

**不存储收益数据**：
- 不存储任何收益数据
- 不提供任何收益预测功能
- 不提供任何交易相关功能

### 主动放弃的机会类型

**收益预测功能**：
- 放弃收益预测功能
- 放弃收益统计功能
- 放弃收益对比功能

**交易信号系统**：
- 放弃交易信号系统
- 放弃买卖时机提示
- 放弃标的推荐功能

**自动化交易决策**：
- 放弃自动化交易决策
- 放弃自动化执行功能
- 放弃跟单功能

**投资建议服务**：
- 放弃投资建议服务
- 放弃资产配置方案
- 放弃买卖决策服务

### 为什么"慢"是策略而不是妥协

**慢但稳意味着可持续性更强**：
- 不依赖市场行情，不受市场波动影响
- 不依赖预测，不受预测准确性影响
- 不踩监管红线，合规风险更低

**慢但稳意味着长期价值更高**：
- 行为数据具有时间价值
- 系统化陪跑具有稀缺性
- 长期行为资产具有独特性

**慢但稳意味着风险更低**：
- 不依赖市场行情，风险更低
- 不依赖预测，风险更低
- 不踩监管红线，风险更低
