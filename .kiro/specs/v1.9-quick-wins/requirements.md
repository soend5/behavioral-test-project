# V1.9 Quick Wins 需求文档

## 概述
本次迭代为 Quick Wins 阶段（1周），聚焦 Admin 导航重组、信息分层、话术优化等快速提升用户体验的改进。

**核心目标**：
- Admin 导航完整度 100%
- 助教话术使用率 +20%
- 配置理解成本降低

---

## 功能模块

### 1. Admin 导航重组

#### 1.1 导航分组
- **现状**: 导航平铺 9 个入口，缺少数据看板、话术库、训练计划
- **需求**: 按业务职能分组，补充缺失入口
- **分组结构**:
  - 运营：数据看板
  - 策略：SOP配置、话术库、训练计划
  - 内容：题库、画像文案、内训手册、方法论
  - 管理：助教账号、系统设置、审计日志
- **终局模型支持**: 否

#### 1.2 验收标准
- [ ] 导航按 4 个分组展示
- [ ] 所有页面可从导航访问
- [ ] 分组标签清晰可见
- [ ] 移动端响应式适配

---

### 2. Admin 首页改为数据看板

#### 2.1 首页重定向
- **现状**: Admin 首页为"总览"页面
- **需求**: 登录后直接重定向到 `/admin/dashboard`
- **终局模型支持**: 否

#### 2.2 验收标准
- [ ] 访问 `/admin` 自动跳转到 `/admin/dashboard`
- [ ] 登录后直达数据看板
- [ ] 无闪烁或延迟

---

### 3. 助教端客户详情页信息分层

#### 3.1 布局重构
- **现状**: 信息平铺，关键信息（画像、阶段、分层标签）不突出
- **需求**: 按重要性分层展示
- **新布局**:
  1. 关键信息卡片置顶（画像类型 + 陪跑阶段 + 分层标签）
  2. 一键话术紧随其后（带"智能推荐"标识）
  3. 陪跑建议精简版
  4. 快速记录入口
  5. 更多信息默认折叠
- **终局模型支持**: 是（阶段：各陪跑阶段；标签逻辑：画像类型、客户分层等）

#### 3.2 验收标准
- [ ] 关键信息首屏可见（移动端）
- [ ] 画像类型、陪跑阶段、分层标签突出显示
- [ ] 一键话术位置上移
- [ ] 次要信息可折叠

---

### 4. 话术面板优化

#### 4.1 位置与标识
- **现状**: 话术面板位置靠下，无明显标识
- **需求**: 
  - 移到页面顶部位置
  - 增加"智能推荐"标记
  - 移动端：大按钮 + 全宽显示
- **终局模型支持**: 否

#### 4.2 使用日志
- **需求**: 复制话术后自动记录使用日志
- **目标**: 话术使用率 +20%

#### 4.3 验收标准
- [ ] 话术面板位于客户详情页顶部区域
- [ ] 显示"智能推荐"标识
- [ ] 复制操作记录到日志
- [ ] 移动端按钮足够大

---

### 5. 待办面板默认展开

#### 5.1 默认状态
- **现状**: 待办列表默认折叠
- **需求**: 改为默认展开
- **终局模型支持**: 否

#### 5.2 验收标准
- [ ] 助教仪表盘待办面板默认展开
- [ ] 用户可手动折叠
- [ ] 状态持久化（可选）

---

### 6. 合规提示措辞优化

#### 6.1 文案更新
- **需求**: 优化各端合规提示文案
- **新文案**:
  - 助教端对话框: "以下为沟通参考，请勿作为投资建议"
  - 结果页: "这是你的行为结构画像，用于和助教对齐下一步"
  - 落地页: "本测评帮你看清操作习惯，不涉及投资建议"
- **终局模型支持**: 否

#### 6.2 验收标准
- [ ] 所有合规提示文案更新
- [ ] 文案清晰、无歧义
- [ ] 符合合规要求

---

### 7. SOP/话术页面文案优化

#### 7.1 术语统一
- **需求**: 清理优化 Admin 策略配置界面文案
- **改动对照**:
  | 改前 | 改后 |
  |------|------|
  | SOP 配置管理 | 陪跑策略配置 |
  | 匹配规则 | 触发条件 |
  | 话术库管理 | 沟通话术库 |
  | 触发标签 | 适用场景 |
- **终局模型支持**: 否

#### 7.2 验收标准
- [ ] SOP 页面标题改为"陪跑策略配置"
- [ ] 话术页面标题改为"沟通话术库"
- [ ] 相关术语统一更新
- [ ] 无入口命名冲突
